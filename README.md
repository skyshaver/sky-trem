# sky-trem
Rhythmic Tremolo Audio Plugin 

SkyTrem is a rhythmic tremolo audio effect plugin developed using the JUCE audio framework

It allows for beat synced tremolo effects using a variety of LFO waveforms

This is a work in progress and the GUI and feature set are incomplete
